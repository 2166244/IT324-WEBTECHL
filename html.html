HTML DOCUMENT STRUCTURE

Composed of three parts:

  • Information about HTML version
  • The Header section
  • The body

HTML (HyperText Mark-up Language) 1990, WWW language for publishing use for creating hypertext documents such as mails, news, documentation, etc. HTML is an application of ISO Standard 8879:1986 Information Processing Text and Office Systems

VERSION HISTORY OF HTML

STANDARDS

    HTML 2.0 (RFC 1866, Nov 1995) T. Berners-Lee and D. Connolly, November 1995. Supports SGML and ISO which both have machine-readabl text.

    HTML 3.2 (W3C Recommendation, Jan 1997) with the compatibility of the HTML 2.0, adds features such as tables, text images, and applets.

    HTML 4.0 (W3C Recommendation, Dec 1999) provides features among users that has disabilities that they can easily access such as style sheets, scripting languages, different multimedia, etc. provides logical structures of documents. conforms with ISO 8879

    HTML 4.0.1 (W3C Recommendation) Subversion of HTML 4.0 is a revision of HTML 4.0 that corrects errors and makes some changes since the previous revision. Different segments of structure and presentation Considerably accessible universally to the Web

    XHTML (W3C Recommendation, August 2002) is part of the extension of HTML4 such as modules and subset. conforms with XML documents XHTML documents relies with XML/HTML DOM

    HTML 5 (W3C Recommendation, October 2014) Supports formatting of web pages, applications, and the Open Web Platform's cornerstone. Supports video and audio even without plugins. supports SVG) and MathML, Ruby annotations

HTML ELEMENT

THE DOCUMENT HEAD

The HEAD element

Compose of the document’s title, keywords and other information but not part of the content of the document.

The TITLE element

This element is not part of the sequence of the text. Example: Page header, window title

The TITLE attribute

This element is useful when you want to link an element with the external stylesheet.

Meta data

Provides specific property and assigns value

Example: property - “Author” value - ”David Ong”

THE DOCUMENT BODY

THE BODY ELEMENT

contains the document's content in many ways.

Element identifiers: the id and class attributes

The id attribute provides the element a distinct identifier The class attribute provides the element one or more names of class/class name.

Block-level and inline elements

Block-level are HTML elements in the body Inline are other also called the text level

Grouping elements: the DIV and SPAN elements

The DIV and SPAN elements, is connected to the id and class attributes which is useful for adding document’s structure.

Headings: The H1, H2, H3, H4, H5, H6 elements

Defines the topic per section of the document

The ADDRESS element

may be used by authors to supply contact information for a document or a major part of a document such as a form.
