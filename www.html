Web Technologies

IP Address

IP address or Internet Protocol address or simply an IP, is a unique address used in identifying a device on the Internet or in a local network.

DNS

DNS or Domain Name System is used in converting domain names in IP or Internet Protocol addresses. DNS Server will covert the URL or Uniform Resource Locator and web address to IP address when it is inputted into the web browser.

Loopback Address

LOOPBACK ADDRESS is a type of Internet Protocol address or IP address used in testing the communication or medium transportation on local network and/or to test network applications.
