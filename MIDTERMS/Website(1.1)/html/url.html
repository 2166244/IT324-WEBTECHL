URL

A Uniform Resource Locator (URL) is a reference of a resource on the Internet. It is used in accessing web resources via Internet by providing identification of the resource location.

Note 1

URL General Syntax

URL is used to locate web resources using the resource location. Different Operations; access, update, replace, find

access method needs specification for URL scheme.

Note 2

The main part of URL

URL contains scheme, colon, and a string Scheme names has a sequence of characters. The letters "a"--"z", digits, and plus sign ("+"), period ("."), and hyphen ("-") are allowed.

Note 3

URL Character Encoding Issues

URL is a sequence of; letters, digits, special characters.

URL depends on the characters being used. characters in US-ASCII character set represents the octets octets that may encode consists of; the character "%", hexadecimal digits ( "0123456789ABCDEF").

The characters "abcdef" is also used in character encodings.

OCTETS MUST BE ENCODED IF; there is no equivalent character from the US-ASCII character set, using a character is unsafe, the character that corresponds to it is reserved within the particular URL scheme.

URLs are only written in characters of US-ASCII coded character set.

Note 4

URL Character Encoding Issues

Different unsafecharacters are; "<" and ">", the quote mark ("""), character "#", character "%" space is also unsafe in URL Other characters that are unsafe: "{", "}", "|", "\", "^", "~", "[", "]", and "`".
