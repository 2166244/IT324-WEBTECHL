    Web Technologies

    Web Technology is the method for different computers to communicate with each another with the use of HTML and multimedia packages. It gives a way for everyone to access different hosted information like websites. It uses HTML or Hypertext Markup language and CSS or Cascading Stylesheet.

    Web Technologies

    +

    IP address or Internet Protocol address or simply an IP, is a unique address used in identifying a device on the Internet or in a local network./p>

    +

    DNS or Domain Name System is used in converting domain names in IP or Internet Protocol addresses. DNS Server will covert the URL or Uniform Resource Locator and web address to IP address when it is inputted into the web browser.

    +

    LOOPBACK ADDRESS is a type of Internet Protocol address or IP address used in testing the communication or medium transportation on local network and/or to test network applications.

    +

    VIRTUAL HOST is used in hosting different domain names.It provides Web services such as server functions and Internet connection services. It is often used in different companies that does not want to avail their own Internet connections and Web servers.

    +

    WAIS or Wide Area Information Server is an Internet System where databases are created in multiple locations of server and kept in track by a directory of a server in a single location.
